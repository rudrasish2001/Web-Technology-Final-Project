ORDERED HTML LIST:


An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

The list items will be marked with numbers by default:


Example:

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk
           
VALUEDESCRIPTION
type="1" The list items will be numbered with numbers (default)
type="A" The list items will be numbered with uppercase letters
type="a" The list items will be numbered with lowercase letters
type="I" The list items will be numbered with uppercase roman numbers
type="i" The list items will be numbered with lowercase roman numbers

Example - Number:

<ol type="1">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk

Example - Uppercase Letters:

<ol type="A">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk

Example - Lowercase Letters:

<ol type="a">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk

Example - Uppercase Roman Numbers:

<ol type="I">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk

Example - Lowercase Roman Numbers:

<ol type="i">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk

Example - Control List Counting:


By default, an ordered list will start counting from 1.If you want to start counting from a specified number,

you can use the start attribute:


<ol start="50">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
  3. Milk

NESTED HTML LISTS

Lists can be nested (list inside list):


Example:

<ol>
  <li>Coffee</li>
  <li>Tea
    <ol>
      <li>Black tea</li>
      <li>Green tea</li>
    </ol>
  </li>
  <li>Milk</li>
</ol>

OUTPUT:

  1. Coffee
  2. Tea
    1. Black tea
    2. Green tea
  3. Milk
